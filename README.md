# 微信小程序开发项目——图片扫描提取文字(OCR)

##完整的一个图片OCR微信小程序项目，采用了百度OCR的API和百度翻译API，实现了拍照，选图，批量图片识别提取文字，表格识别，图片剪裁，支持分享，翻译，校对，记录识别历史等功能，还使用了微信小程序云函数进行识别后文字的安全鉴定。
